Travel Focus App

집중을 시작하는 시간도,
여행처럼.

집중 여행 서비스는 화이트노이즈와 여행 서사를 결합해, 사용자가 더 자연스럽게 몰입 상태로 들어가도록 설계한 앱입니다. 타이머를 켜는 대신 누구와 떠날지 고르고, 얼마나 떠날지 선택하며 집중을 시작합니다.

집중의 시작을 여행의 출발감으로 바꾸는 앱 경험
비행기, 관광지, 음식점, 숙소 흐름에 맞춘 장면 설계
공간 기반 화이트노이즈와 시네마형 세션 화면
앱 화면 보기 기획 문서 보기
누구와 여행을 떠날까요? 함께 여행할 친구를 선택하고, 오늘의 집중을 여행의 리듬으로 시작합니다.
떠나볼까요?

핵심 경험은 단순합니다.

사용자는 복잡한 설정 없이 캐릭터를 고르고, 여행 시간을 정하고, 장면과 사운드가 자동으로 이어지는 흐름 안에서 집중을 시작합니다.

01

여행 친구 선택

첫 화면에서 함께 떠날 캐릭터를 선택하며, 특정 여행 세계관에 자연스럽게 입장합니다.

02

여행 시간 선택

국내여행부터 남극 여행까지, 집중 시간을 여행 거리처럼 해석해 선택합니다.

03

장면 기반 몰입

비행기에서 시작해 관광지, 음식점, 숙소로 이어지는 장면이 세션 흐름을 만듭니다.

04

도착형 종료

집중 종료는 단순한 타이머 완료가 아니라, 한 번의 여행이 끝나는 감정으로 마무리됩니다.

서비스 구조는 감성적이지만, 운영은 명확합니다.

캐릭터별 공간 비율, 시작과 종료 규칙, 공간별 행동 세트, 배경 이미지 전환 규칙까지 실제 출시를 고려한 수준으로 정리했습니다.

캐릭터 구조

5명의 오마주형 캐릭터를 기반으로 외형, 공간 비율, 공간 등급 확률을 분리해 설계했습니다.

세션 리듬

비행기 10% 시작, 숙소 15% 종료를 고정하고 중간 구간은 캐릭터별 공간 확률로 구성합니다.

장면 자산

비행기, 관광지, 음식점, 숙소 등 서비스 단계별 배경 이미지를 반영하고 일부 구간은 랜덤 노출로 처리합니다.

실제 앱 화면 흐름을 바로 확인할 수 있습니다.

첫 진입 화면부터 시간 선택, 비행기 세션, 관광지, 음식점, 숙박, 종료 화면까지 한 흐름으로 연결된 목업입니다.

HOME

첫 진입 화면

공항 이미지와 하단 카드 구조를 통해, 감정적으로 진입하면서도 앱 서비스답게 시작하는 화면입니다.

TIME

시간 선택 화면

집중 시간을 숫자가 아닌 여행 거리로 번역해, 선택 행위 자체가 더 서비스다워지도록 설계했습니다.

SESSION

세션 진행 화면

비행기, 관광지, 음식점, 숙박 화면이 이어지고 일부 장면은 랜덤 배경으로 교체돼 반복 피로를 줄입니다.

FINISH

종료 화면

내 방으로 돌아온 듯한 마무리 이미지를 사용해, 도착형 종료 경험과 기록형 요약을 함께 전달합니다.

앱 화면과 기획 문서를 함께 확인해보세요.

목업으로는 실제 흐름을, 문서로는 기획 의도와 정책 구조를 빠르게 파악할 수 있습니다.
